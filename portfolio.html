Portal Bolso Digital - Plataforma web

BuscaCEP - Consulta de endereços

Site desenvolvido para consultar endereços a partir de um CEP informado pelo usuário, com preenchimento automático dos dados retornados pela API. O projeto também permite busca por rua, cidade e UF.

  • Consulta de endereço por CEP usando BrasilAPI.
  • Busca por rua, cidade e UF usando ViaCEP.
  • Carregamento de estados e cidades com dados do IBGE.
  • Interface responsiva com campos de formulário e botões de busca.

Tecnologias: HTML5, CSS3, JavaScript ES6, Fetch API, BrasilAPI, ViaCEP e API de localidades do IBGE.
